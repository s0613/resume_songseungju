## 코어 5선

### 1. MONE — 선상 면세점 통합 관리 시스템
실제 운영 비즈니스에 임베드되어 예약·주문·정산·인사급여 등 **13개 도메인**을 모놀리식 구조로 통합한 풀스택 시스템. 복잡한 현장 요구사항을 엔드투엔드로 납품한, 가장 FDE다운 사례.

- **기간:** 2026.01 ~ 현재
- **스택:** Django · Python · DRF · Next.js · TypeScript · PostgreSQL
- **링크:** https://www.songseungju.dev/portfolio/mone

### 2. MediVu — PACS 연동 AI 판독문 자동 생성
병원의 **레거시 의료 시스템(PACS) 연동 문제**를 AI로 해결한 솔루션. 고객의 실환경 제약 안에서 통합을 설계·구현한, 전형적인 "지저분한 실환경 임베드" 케이스.

- **기간:** 2025.03 ~ 2025.10
- **스택:** Next.js · TypeScript · MCP · PDF De-identification
- **링크:** https://www.songseungju.dev/portfolio/medivu

### 3. Totaload — 중고차 수출 디지털 인증서 자동 발급 SaaS
도메인 규정을 **AI 판독 + 규정 매핑**으로 자동화한 원스톱 솔루션. 규제·도메인 복잡도가 높은 영역을 제품으로 풀어낸 사례.

- **기간:** 2025.07 ~ 현재
- **스택:** Spring Boot · Python AI · AWS · OCR · PDF · 전자서명
- **링크:** https://www.songseungju.dev/portfolio/totaload

### 4. Cogmo 안녕 — AI 기반 인지건강 측정·관리 플랫폼
모바일 앱·보호자 대시보드·백엔드 풀스택 + **Gemini AI** 연동, 실시간 모니터링·PDF 리포트 자동화까지 단독 구현. AI 통합 납품력을 증명하는 사례.

- **기간:** 2025.11 ~ 2026.01
- **스택:** Flutter · Spring Boot · Next.js · Gemini AI · AWS · WebSocket
- **링크:** https://www.songseungju.dev/portfolio/cogmo

### 5. Upflow AX — SI 사업 전 주기 관리 플랫폼 *(자체 제품)*
제안부터 정산까지 하나의 워크스페이스로 통합. DDD 도메인 캔버스·RTM·손익 관리. **"FDE 방식의 일하는 방법론을 제품으로 만든 것"** — 회사 대표작 포지셔닝에 최적.

- **링크:** https://www.songseungju.dev/portfolio/upflowax
